# 라이브러리를 익히고 사용하라

### `표준 라이브러리`를 사용할때 장점
+ 표준 라이브러리를 사용하면 선배 프로그래머들의 경험을 활용할 수 있다.
+ 표준 라이브러리를 사용하는 두번째 이점은, 비즈니스 로직에 집중이 가능하다.
+ 표준 라이브러리는 지속적으로 업데이트되기 때문에, 성능이 지속해서 개선된다.
+ 기능이 점점 많아진다. 
+ 라이브러리를 이용할때, 표준이기때문에 다른사람에게도 익숙하다.
